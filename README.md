# Dataset
Dataset yang digunakan adalah kumpulan komentar pengguna TikTok menenai Ibu Kota Nusantara (IKN). Data dikumpulkan dari platform TikTok dan telah melalui proses pelabelan sentimen secara manual.
Dataset terdiri dari 1472 komentar, yang masing-masing diklasifikasikan ke dalam dua kateori sentiman yaitu:
Kategori Sentimen
|---------|
|Sentimen Positif adalah komentar yang mendukung, memberikan apresiasi, atau memiliki opini baik terhadap IKN|
|Sentimen Negatif adalah komentar yang mengandung kritik, penolakan, atau opini kurang baik terhadap IKN|

Dataset ini mempresentasikan opini publik yang beragam dan bersifat tidak terstruktur, sehingga cocok digunakan untuk studi klasifikasi teks.


## Task
Proyek ini  bertujuan membangun model klasifikasi teks untuk memprediksi sentimen komentar TikTok tentang IKN menggunakan pendekatan Machine Learning dan Deep Learning.
Adapun task yang dilakukan dalam proyek ini meliputi:
|No.  | Task    |
|-----|---------|
| 1. | Preprocessing Data (Pembersihan dan Normalisasi Teks) |
| 2. | Eksplorasi Data |
| 3. | Feature Extraction (mengubah data teks menjadi numerik) |
| 4. | Pembangunan Model |
| 5. | Evaluasi Model |
